경찰청의 주요 서비스를 메인 화면에 직관적으로 배치하여 이용자가 필요한 기능을 한눈에 확인할 수 있도록 접근성을 강화한 공공기관 웹사이트입니다. 모던한 퍼블리싱 기법을 적용하여 신뢰성과 청렴한 이미지를 시각적으로 표현했습니다.

HTML5 CSS3 JavaScript (ES6+) CSS Variables Flexbox & Grid 반응형 디자인

핵심 기능

  • 메인 슬라이더 (자동 재생, 터치 스와이프 지원)
  • 빠른 메뉴 (Font Awesome 아이콘, 호버 효과)
  • 드롭다운 서브메뉴 (그리드 레이아웃)
  • 열린청장실 & 관련사이트 영역
  • 모바일 햄버거 메뉴
  • 팝업 모달 (ESC 키 지원)

디자인 & 기술

  • CSS Variables를 활용한 디자인 시스템 구축
  • 블루 계열 색상으로 신뢰성과 청렴함 표현
  • Flexbox & Grid를 활용한 모던 레이아웃
  • 반응형 디자인 (768px, 992px, 1200px 브레이크포인트)
  • 접근성 개선 (ARIA 속성, 시맨틱 HTML5)
  • 부드러운 애니메이션과 트랜지션 효과
