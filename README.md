# Hyehyeon's Portfolio Site

개인 포트폴리오를 소개하는 웹사이트입니다. Publishing, Frontend, Backend 프로젝트들을 카테고리별로 확인할 수 있습니다.

## 프로젝트 개요

웹 개발자로서의 기술 스택과 완성한 프로젝트들을 한눈에 보여주기 위한 포트폴리오 사이트입니다. 반응형 레이아웃과 모달 기반 상세 뷰를 통해 각 프로젝트의 정보를 효과적으로 전달합니다.

## 주요 기능

### About Me 섹션
- 프로필 정보 (이름, 이메일, GitHub 링크)
- 기술 스택을 4개 카테고리로 분류하여 표시
  - Frontend (React, Next.js, Vue.js 등)
  - UI & Experience (Figma, 반응형 레이아웃 등)
  - Backend (Java, Spring Boot, MySQL 등)
  - Collaboration (Git, GitHub Flow, Jira 등)

### 포트폴리오 갤러리
- 카테고리 필터링 기능 (All / Publishing / Frontend / Backend)
- 프로젝트 카드 호버 효과
- 모달 팝업을 통한 상세 정보 제공
  - 제작 기간, 제작물, 제작 의도
  - Style Guide 이미지
  - 시안 이미지
  - 사이트 바로가기 링크

### 포함된 프로젝트

**Publishing Projects**
- Molly's Pet Shop
- 대전지방경찰청
- 맥심 브랜드

**Frontend Projects**
- 맛콩맛집 (공공데이터 + 카카오 지도 연동)

**Backend Projects**
- 감정 블로그 플랫폼 (Spring Boot + PostgreSQL)

## 특징

- 📱 **반응형 디자인**: 모바일, 태블릿, 데스크톱 모든 환경 지원
- 🎨 **깔끔한 UI**: Bootstrap 기반의 모던한 디자인
- 🔍 **필터링 기능**: 프로젝트를 카테고리별로 쉽게 탐색
- 💬 **상세 모달**: 각 프로젝트의 세부 정보를 팝업으로 제공
- 🎯 **직관적인 네비게이션**: 스크롤 기반의 원페이지 레이아웃
